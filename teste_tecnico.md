# Desafio de Vaga de Desenvolvedor(a) Full Stack

## Descrição do Desafio

Você foi contratado(a) por uma empresa fictícia chamada **TechX** para desenvolver uma aplicação web de gerenciamento de tarefas (**To-Do List**). A empresa precisa de uma solução simples e eficiente para que seus funcionários possam organizar suas tarefas diárias.

---

## Requisitos Técnicos

### Frontend

* Desenvolva a interface de usuário utilizando **Angular**.
* A página inicial deve exibir todas as tarefas cadastradas.
* Permita a adição, edição e remoção de tarefas.
* Implemente funcionalidades de marcação de tarefas como concluídas ou não.

### Backend

* Crie uma API RESTful utilizando **Node.js** e **TypeScript**.
* A API deve ser capaz de receber requisições para criar, listar, atualizar e excluir tarefas.
* Utilize um banco de dados **MySQL** para armazenar os dados das tarefas.

### Versionamento de Código

* Utilize o **Git** para versionar seu código.
* Faça commits incrementais e utilize mensagens de commit descritivas.

---

## Desafio Extra (Opcional)

* Implemente uma funcionalidade de autenticação de usuários utilizando **JWT (JSON Web Token)**.
* Integre o backend com um banco de dados **NoSQL**, como o **MongoDB**, para armazenar informações adicionais sobre as tarefas.

---

## Critérios de Avaliação

### Funcionalidade

A aplicação deve cumprir todos os requisitos funcionais descritos acima.

### Qualidade de Código

Avaliaremos:

* Estrutura do código;
* Boas práticas de programação;
* Modularização;
* Legibilidade.

### Utilização das Tecnologias Especificadas

Deve-se utilizar as tecnologias requeridas e demonstrar compreensão básica de cada uma delas.
